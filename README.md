# Site Epigenics - Starter

Projeto Vite + React para o site da Epigenics.

## Como usar

1. Copie todos os arquivos para o repositório no GitHub/Codespaces.
2. Coloque as imagens dentro de `public/assets/` respeitando as pastas e nomes combinados.
3. No terminal, rode:

```bash
npm install
npm run dev
```

## Onde trocar WhatsApp

Procure no arquivo `src/App.jsx` por:

```txt
5541999999999
```

Troque pelo número correto com DDD.
